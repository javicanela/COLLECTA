# CLAUDE.md

Codex es la herramienta principal de este repositorio.

Claude Code puede asistir desde terminal, pero debe seguir la misma fuente
operativa que Codex:

1. Lee `AGENTS.md`.
2. Lee `docs/PLAN_DEFINITIVO_COLLECTA.md`.
3. Si el trabajo toca importacion, lee `docs/specs/smart-import-super-identifier.md`.
4. Si hay conflicto documental, manda `AGENTS.md`.

No crees ni restaures carpetas `.claude/` o `.ai-*` dentro del repo. Si necesitas
configuracion local, mantenla fuera del proyecto y documenta el bloqueo.
